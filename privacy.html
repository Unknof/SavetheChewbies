Privacy & contact

This is an early version. These notes will be refined as the project grows.

Privacy

This site is built to verify donations without screenshots. We try to keep personal data minimal.

What we store

  • Verification code cookie: A short-lived cookie is stored to remember your most recent verification code so the Verify page can pre-fill it.
  • Verification status: Server-side status for that code (pending/verified/cancelled + timestamps).
  • Webhook audit log: A minimal log of webhook events for debugging/auditing. This may include donor display name and donation message if Tiltify sends it (and if the donor provided it).

We don’t handle payment information. Donations happen on Tiltify.

Retention

Verification records/logs are kept only as long as needed to operate and debug the verification flow.

Deletion requests

Email the address below with your verification code and we can remove the related server-side record.
